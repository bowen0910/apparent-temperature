體感溫度計算機

計算公式 1. **低溫風寒公式 (T ≤ 15°C)**：
T_cold = T - 0.7*V_eff - 0.12*(H/100)*(T-10)

2. **高溫公式 (T ≥ 27°C)**：
T_high = T + 0.33*(H/100)*T

3. **線性過渡區公式 (15°C < T < 27°C)**：
T_app = w_cold*T_cold + (1-w_cold)*T_high ，其中 w_cold = (27-T)/12

4. **日照修正公式**：
- 低溫區：夜間 -0.2 / 陰天 0 / 多雲 -0.1 / 晴天 0.6 / 強烈日照 1.2
- 高溫區強烈日照上限限制 +2°C
T_app = T_app + 日照修正值

5. **降雨及降雪影響溫度公式**：
F_precip = 降雨或降雪係數 (依強度及溫度區間)

6. **最終體感溫度**：
T_final = T_app + F_precip ，四捨五入至小數點第一位

本網站計算之體感溫度僅供參考，實際感受會因個人體質、衣著、活動量、日照、降水及風向等等因素而有所差異。
