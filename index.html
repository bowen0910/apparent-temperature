體感溫度計算機

無法判斷時，建議選擇「多雲」。
計算公式 【一、單位轉換】
- km/h → m/s：V = km/h ÷ 3.6
- 節 (kt) → m/s：V = kt × 0.514444
- 蒲式風級 → m/s：V = 0.836 × B3/2
- 華氏 → 攝氏：T(°C) = (T(°F) − 32) × 5 / 9

【二、有效風速】
- 若有陣風：Veff = (平均風速 + 陣風) ÷ 2
- 若無陣風：Veff = 平均風速

【三、低溫風寒公式（T ≤ 15°C）】
Tcold = T − 0.7 × Veff − 0.2 × (H / 100) × (T − 10)

【四、高溫熱指數（T ≥ 27°C，簡化式）】
Thigh = T + 0.33 × (H / 100) × T

【五、線性過渡區（15°C < T < 27°C）】
權重：w = (27 − T) / 12
Tapp = w × Tcold + (1 − w) × Thigh

【六、日照修正】
- 低溫區（≤15°C）：0.0 / −0.5 / −0.2 / +0.5 / +1.0°C
- 過渡區（15–27°C）：0.0 / +0.5 / +1.5 / +2.5 / +4.0°C
- 高溫區（≥27°C）：0.0 / +0.5 / +1.0 / +1.3 / +2.0°C

【七、結果】
最終體感溫度四捨五入至小數點第一位。

體感溫度僅供參考，實際感受會因個人體質、衣著與活動量而有所不同。
